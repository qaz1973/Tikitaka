# Tikitaka 进阶说明

这是 Tikitaka 配置工具的中文进阶说明网站。页面由 `tools/sync-manual.js` 根据录屏目录自动同步生成，并为每个录屏补充下一步索引。站内视频保持原始 MP4 不动，在网页层统一播放速度。

首页 `index.html` 是唯一发布的进阶功能目录，覆盖全部视频。页面默认只展示照做步骤，检查点和注意事项按需展开。

## 在线说明网站

```text
https://qaz1973.github.io/Tikitaka/
```

## 同步录屏

默认录屏来源：

```text
D:\2\说明书视频文件部分
```

重新同步：

```powershell
node tools/sync-manual.js
```

同步后会更新 `index.html`、`README.md`，并同步 `videos/`。

也可以指定其他来源目录：

```powershell
node tools/sync-manual.js "D:\2\说明书视频文件部分"
```

## 视频展示规则

- 源视频不重编码，保留在原录屏目录。
- 页面滚动到某个步骤时，只自动播放当前步骤视频。
- 视频静音、无控制条、循环播放，站内固定按 0.5 倍播放。

## 当前目录

1. 基础设置 / 设备预设 - 按设备型号套用基础配置，视频演示的是打开设备预设并选择对应型号。
2. 基础设置 / 布局预设 - 根据实际面板布局快速生成按键位置，视频演示的是选择布局并套用。
3. 基础设置 / 按键功能调整 - 点击画布上的按键，修改它在手柄里的功能。
4. 基础设置 / 单键行程触发设置 - 只调整一个磁轴按键的总行程和触发行程，视频演示的是点按键后在弹窗里修改数值。
5. 基础设置 / 批量行程触发设置 - 在“更多”菜单里批量填写 Travel 和 Actuation，视频演示的是普通界面也能看到的批量设置。
6. 基础设置 / 单键 RT 设置 - 为单个磁轴按键开启 Rapid Trigger，并设置按下和抬起灵敏度。
7. 基础设置 / 批量 RT 设置 - 对多个磁轴按键统一应用 Rapid Trigger 参数。
8. 基础设置 / 按键校准 - 校准磁轴按键的空闲点和最大按下点，让行程、触发和 RT 判断有可靠基准。
9. 基础设置 / 保存校准结果 - 校准完成后把新的磁轴参数写入设备，避免刷新或断电后丢失。
10. 基础设置 / RT 与行程测试 - 在测试状态下检查按键状态、行程读数和 Rapid Trigger 响应是否正常。
11. 基础设置 / 保存配置到设备 - 把当前页面的设置写入设备存储。
12. 基础设置 / 控制器模式切换 - 切换设备启动后的输入模式，例如 XInput、Switch、DInput、Keyboard、PS4/PS5、Xbox 等。
13. 基础设置 / SOCD 模式切换 - 设置相反方向同时输入时的清理规则。
14. 进阶 / 切换用户模式 - 切换普通/专业界面，显示更多布局、灯光、导入导出和高级维护工具。
15. 进阶 / 导入用户布局 - 导入自定义布局中的部分配置，用于复用已整理好的按键位置或配置片段。
16. 进阶 / 按键通道与引脚分配 - 为按键分配磁轴通道或 GPIO 引脚，解决硬件线路和画布按钮之间的对应关系。
17. 进阶 / 添加按键 - 在专业布局中新增需要配置的按键元素。
18. 进阶 / 屏幕定义与布局保存 - 修改 OLED/屏幕布局中的按钮元素，绑定通道并保存自定义显示布局。
19. 进阶 / 按键灯光顺序设置 - 设置 RGB 按键灯在灯带链路中的顺序，影响流水、渐变和按键灯效对应关系。
20. 进阶 / 按键灯光颜色设置 - 为单个或多个按键设置普通状态和按下状态颜色。
21. 进阶 / 氛围灯配置或预设 - 为外壳或设备内部的 Case RGB/氛围灯添加配置，或套用已有预设。
22. 进阶 / 氛围灯颜色设置 - 调整氛围灯的默认颜色或预设颜色。
23. 进阶 / 自动校准 - 开机或连接后预留约 3 秒初始化时间，自动读取当前磁轴空闲状态并更新校准补偿；用于多平台切换时保证输入稳定，一般保持默认开启。
