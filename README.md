# Tikitaka 使用说明

这是 Tikitaka 配置工具的中文说明网站，包含设备预设、布局预设、普通设置、磁轴校准和进阶设置的文字说明与录屏演示。

## 在线说明网站

仓库上传后可以使用 GitHub Pages 发布：

1. 打开 GitHub 仓库 `Settings`。
2. 进入 `Pages`。
3. `Source` 选择 `Deploy from a branch`。
4. `Branch` 选择 `main`，目录选择 `/root`。
5. 保存后，网站地址通常是：

```text
https://qaz1973.github.io/Tikitaka/
```

## 内容目录

- 预设使用
  - 设备预设：根据购买设备型号快速设置分配。
  - 布局预设：根据购买的型号快速分配 GPIO 或者霍尔通道。
- 普通设置
  - 修改功能：修改常规按键功能和基础配置。
- 磁轴校准
  - 测试按键状态。
  - 校准按键：点击校准开始按键校准，点击要校准的按键，缓慢按压 3 次后完成。
  - 批量 RT 行程触发设置。
  - 单独设置 RT 和行程。
- 进阶设置
  - 切换进阶界面。
  - 添加按键。
  - 分配按键：在模板上设置 GPIO 或者磁轴通道。
  - 定义按键灯颜色。
  - 定义外壳灯颜色。
  - 自动校准：用于防止环境变化导致偏移和误触发，开启后连接设备初始化会较为缓慢。

## 本地查看

直接打开 `index.html` 即可查看说明网站。
