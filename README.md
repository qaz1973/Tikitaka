# Tikitaka 使用说明

这是 Tikitaka 配置工具的中文说明网站。说明页面由 `tools/sync-manual.js` 根据录屏目录自动同步生成。

## 在线说明网站

GitHub Pages 地址：

```text
https://qaz1973.github.io/Tikitaka/
```

## 同步录屏

默认录屏来源：

```text
D:\2\说明书视频文件部分
```

重新同步：

```powershell
node tools/sync-manual.js
```

也可以指定其他来源目录：

```powershell
node tools/sync-manual.js "D:\2\说明书视频文件部分"
```

## 当前步骤

1. 基础设置 / 设备预设 - 按设备型号套用基础配置。
2. 基础设置 / 布局预设 - 按布局型号快速分配 GPIO 或霍尔通道。
3. 基础设置 / 按键功能调整 - 选择按键后修改对应功能。
4. 基础设置 / 行程触发设置 - 单独设置 - 单独调整一个按键的触发行程。
5. 基础设置 / 行程触发设置 - 批量设置 - 批量应用多个按键的触发行程。
6. 基础设置 / RT功能设置 - 单独设置 - 单独设置 RT 下压和释放参数。
7. 基础设置 / RT功能设置 - 批量设置 - 批量应用 RT 参数。
8. 基础设置 / 按键校准 - 按提示完成磁轴按键校准。
9. 基础设置 / 按键校准 - 保存配置 - 校准完成后保存配置。
10. 基础设置 / RT行程校准测试 - 检查 RT、行程和按键状态是否正常。
11. 基础设置 / 保存配置 - 设置完成后写入设备。
12. 基础设置 / 控制器模式切换 - 切换设备输入或控制器模式。
13. 基础设置 / SOCD模式切换 - 选择 SOCD 清理模式。
14. 进阶 / 切换用户模式 - 进入进阶或用户模式界面。
15. 进阶 / 定制用户布局部分导入 - 导入自定义用户布局。
16. 进阶 / 按键通道引脚分配 - 给按键分配 GPIO 或磁轴通道。
17. 进阶 / 屏幕定义 - 修改通道添加按键保存 - 在屏幕布局中修改通道、添加按键并保存。
18. 进阶 / 添加按键 - 在进阶布局中添加需要配置的按键。
19. 进阶 / 灯光颜色设置 - 按键灯光设置 - 灯光设置顺序设置 - 设置按键灯顺序和灯光效果。
20. 进阶 / 灯光颜色设置 - 按键灯光设置 - 按键颜色设置 - 设置按键灯颜色。
21. 进阶 / 灯光颜色设置 - 氛围灯设置 - 添加配置或者使用预设 - 添加氛围灯配置或套用预设。
22. 进阶 / 灯光颜色设置 - 氛围灯设置 - 颜色设置 - 设置氛围灯颜色。
23. 进阶 / 自动校准 - 开启或关闭自动校准。
