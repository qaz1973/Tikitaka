基础流程

先把设备配置到能正常使用。

设备预设

按设备型号套用基础配置，视频演示的是打开设备预设并选择对应型号。

照做

  1. 进入磁轴配置页面，打开“设备预设”。
  2. 选择与你设备型号一致的预设。
  3. 确认选择后套用预设。
检查点和注意事项

完成后确认

  • 按键数量和实际面板一致。
  • 设备预设没有把不需要的 GPIO、LED 或外设配置写入当前设备。

已有可用配置时先备份；第一次配置可以直接按视频套用。如需验证按钮数量或画布布局，请先切换到专业模式。

下一步：布局预设

布局预设

根据实际面板布局快速生成按键位置，视频演示的是选择布局并套用。

照做

  1. 进入磁轴配置页面，打开“布局预设”。
  2. 选择与外壳或键位排列一致的布局。
  3. 套用后看画布上的按钮位置是否接近你的设备。
  4. 如果只是键位功能不对，继续看“按键功能调整”。
检查点和注意事项

完成后确认

  • 方向键、功能键和辅助键位置清楚可辨。
  • 布局不会覆盖你已经调整好的自定义位置。

普通用户优先用布局预设；只有按键对应不上时，再看进阶里的通道分配。

下一步：按键功能调整

按键功能调整

点击画布上的按键，修改它在手柄里的功能。

照做

  1. 在磁轴配置画布上选择一个按键。
  2. 在右侧属性面板中选择“功能”。
  3. 按需要改成方向、B1/B2、L1/R1、S1/S2、A1/A2、Fn 等动作。
  4. 改完后继续看测试或保存配置。
检查点和注意事项

完成后确认

  • 画布上的标签和你的目标键位一致。
  • 进入测试模式后按下按键，触发状态和显示的动作一致。

只改按键功能时，不需要进入专业模式。

下一步：单键行程触发设置

单键行程触发设置

只调整一个磁轴按键的总行程和触发行程，视频演示的是点按键后在弹窗里修改数值。

照做

  1. 选择一个 HE 按键。
  2. 切换到“行程”工具。
  3. 在弹出的行程设置窗口里修改 Travel。
  4. 再修改 Actuation。
检查点和注意事项

剩余动作

  1. 改完后关闭窗口，继续看测试或保存配置。

完成后确认

  • 轻按到目标深度时按键开始触发。
  • 完全释放后不会残留触发状态。

Actuation 数值越小越早触发；如果误触，调大一点再测试。

下一步：批量行程触发设置

批量行程触发设置

在“更多”菜单里批量填写 Travel 和 Actuation，视频演示的是普通界面也能看到的批量设置。

照做

  1. 打开顶部“更多”菜单。
  2. 保持或切到“普通”界面。
  3. 在批量设置里填写行程和触发行程。
  4. 点击“批量应用行程/触发”。
检查点和注意事项

剩余动作

  1. 继续看 RT 设置、测试或保存配置。

完成后确认

  • 同组按键触发手感一致。
  • 没有把不需要调整的按键一起选中。

批量应用会把所选按键改成同一组行程参数，适合 WASD、方向键等需要统一手感的按键。

下一步：单键 RT 设置

单键 RT 设置

为单个磁轴按键开启 Rapid Trigger，并设置按下和抬起灵敏度。

照做

  1. 选择一个 HE 按键。
  2. 切换到“RT”工具。
  3. 打开 Enable Rapid Trigger。
  4. 设置 RT Down 和 RT Up。
检查点和注意事项

剩余动作

  1. 进入测试模式，连续轻按和释放，确认不会误触或粘连。

完成后确认

  • 按下时响应足够快。
  • 轻微抖动时不会反复误触。

RT 数值越小越灵敏，也越依赖轴体和传感器稳定性。

下一步：批量 RT 设置

批量 RT 设置

对多个磁轴按键统一应用 Rapid Trigger 参数。

照做

  1. 选择需要统一 RT 手感的多个 HE 按键。
  2. 切换到“RT”工具并开启 Rapid Trigger。
  3. 填写 RT Down 和 RT Up。
  4. 批量应用后在测试模式中逐个按键检查。
检查点和注意事项

剩余动作

  1. 确认后保存配置。

完成后确认

  • 同组按键的触发和复位速度一致。
  • 没有把辅助键或不需要 RT 的按键一起应用。

批量 RT 适合方向键和常用动作键，不建议在未测试前直接套到所有按键。

下一步：按键校准

按键校准

校准磁轴按键的空闲点和最大按下点，让行程、触发和 RT 判断有可靠基准。

照做

  1. 进入按键校准或批量校准入口。
  2. 保持所有按键松开，等待空闲采样完成。
  3. 按界面提示逐个按下目标按键到最大行程。
  4. 等待系统记录按下点后松开。
检查点和注意事项

剩余动作

  1. 全部完成后看页面状态，再进入保存校准结果。

完成后确认

  • 空闲状态下按键不触发。
  • 完全按下时状态稳定触发。

校准时不要压着其他按键。

下一步：保存校准结果

保存校准结果

校准完成后把新的磁轴参数写入设备，避免刷新或断电后丢失。

照做

  1. 确认所有需要校准的按键已经完成。
  2. 点击保存或保存校准结果。
  3. 等待保存完成提示。
  4. 保存后再做一次 RT 与行程测试。
检查点和注意事项

完成后确认

  • 页面提示保存成功。
  • 刷新页面或重新进入后，校准状态仍然正常。

未保存就离开页面、刷新或重启，校准结果可能不会写入设备。

下一步：RT 与行程测试

RT 与行程测试

在测试状态下检查按键状态、行程读数和 Rapid Trigger 响应是否正常。

照做

  1. 打开测试模式。
  2. 逐个按下需要检查的按键。
  3. 观察触发状态、行程读数和释放响应。
  4. 发现异常时返回校准、行程或 RT 设置重新调整。
检查点和注意事项

剩余动作

  1. 测试通过后保存最终配置。

完成后确认

  • 按下、保持、释放三个阶段都稳定。
  • RT 开启的按键能随释放和再按快速响应。

测试模式读取运行时状态，不等于已经保存；参数确认后仍要执行保存。

下一步：保存配置到设备

保存配置到设备

把当前页面的设置写入设备存储。

照做

  1. 确认预设、按键、行程、RT、SOCD 或模式设置都已经检查完。
  2. 点击保存配置。
  3. 等待保存完成提示。
  4. 如果页面提示需要重启，再从顶部导航选择重启方式。
检查点和注意事项

完成后确认

  • 页面出现保存成功或进度完成提示。
  • 重新进入页面后关键参数仍然保留。

保存、重启、恢复默认都会改变设备状态，请确认后再执行。

下一步：控制器模式切换

控制器模式切换

切换设备启动后的输入模式，例如 XInput、Switch、DInput、Keyboard、PS4/PS5、Xbox 等。

照做

  1. 进入设置或磁轴页顶部的输入模式入口。
  2. 选择目标控制器模式。
  3. 如果页面提示需要额外认证或连接手柄，按页面提示处理。
  4. 保存设置。
检查点和注意事项

剩余动作

  1. 必要时重启到 Controller 模式，让系统重新识别设备。

完成后确认

  • 电脑或主机识别出的设备类型符合目标模式。
  • 按键测试工具中输入正常。

切换模式后电脑或主机可能会重新识别设备，短暂断开是正常现象。

下一步：SOCD 模式切换

SOCD 模式切换

设置相反方向同时输入时的清理规则。

照做

  1. 进入磁轴配置页或设置页。
  2. 找到 SOCD 清理模式。
  3. 按需要选择 Up Priority、Neutral、Last Win、First Win 或 Off。
  4. 保存配置并测试左右、上下同时输入。
检查点和注意事项

完成后确认

  • 同时按下相反方向时输出符合你选择的规则。
  • 目标游戏或比赛规则允许当前 SOCD 模式。

Off/Bypass 表示不清理相反方向输入，合规性需要自行确认。

下一步：切换用户模式

进阶功能

需要时再看，普通用户可以先跳过。

切换用户模式

切换普通/专业界面，显示更多布局、灯光、导入导出和高级维护工具。

照做

  1. 在磁轴配置页面打开“更多”或界面模式入口。
  2. 选择普通模式或 Pro/专业模式。
  3. 进入专业模式后再进行布局解锁、添加按键、灯光工具或自动校准表操作。
  4. 完成后可切回普通模式，减少误操作。
检查点和注意事项

完成后确认

  • 专业模式下可以看到解锁、添加按键、OLED 布局、LED 工具等入口。
  • 普通模式下危险或复杂入口被隐藏。

专业模式会暴露更多会改变布局和硬件映射的功能，建议只在需要时打开。

下一步：导入用户布局

导入用户布局

导入自定义布局中的部分配置，用于复用已整理好的按键位置或配置片段。

照做

  1. 进入专业模式。
  2. 打开导入入口，选择布局或配置文件。
  3. 只导入需要的部分，避免覆盖无关设置。
  4. 导入后看画布位置和按键功能是否符合预期。
检查点和注意事项

剩余动作

  1. 确认后保存。

完成后确认

  • 导入后的布局和设备实际面板一致。
  • 没有把旧设备的通道或 GPIO 配置误带入当前设备。

跨设备导入前建议先备份当前配置；不确定时只导入布局相关内容。

下一步：按键通道与引脚分配

按键通道与引脚分配

为按键分配磁轴通道或 GPIO 引脚，解决硬件线路和画布按钮之间的对应关系。

照做

  1. 进入专业模式并选择目标按键。
  2. 在属性面板中选择该按键对应的输入来源。
  3. 按视频给按键选择对应通道或引脚。
  4. 分配动作功能。
检查点和注意事项

剩余动作

  1. 保存前到测试演示里按一次实体按键确认对应关系。

完成后确认

  • 按下实体按键时，画布上对应按钮被触发。
  • GPIO 引脚没有覆盖插件或外设正在占用的引脚。

这是专业设置；普通用户只有在按键对应不上、布局预设无法解决时再使用。

下一步：添加按键

添加按键

在专业布局中新增需要配置的按键元素。

照做

  1. 切换到专业模式并解锁画布。
  2. 选择添加按键工具和按钮尺寸。
  3. 在画布目标位置点击添加。
  4. 为新按键选择输入来源。
检查点和注意事项

剩余动作

  1. 设置动作功能、行程或灯光后保存。

完成后确认

  • 新增按键位置不遮挡其他元素。
  • 新增按键已经绑定通道/引脚和功能。

只添加画布按钮还不能直接使用，需要继续看“按键通道与引脚分配”。

下一步：屏幕定义与布局保存

屏幕定义与布局保存

修改 OLED/屏幕布局中的按钮元素，绑定通道并保存自定义显示布局。

照做

  1. 进入 OLED 布局或屏幕定义编辑器。
  2. 解锁布局编辑。
  3. 添加椭圆、方形或按钮元素。
  4. 为元素选择要显示的按键或方向。
检查点和注意事项

剩余动作

  1. 检查位置、大小和显示层级。
  2. 保存自定义布局，并回到显示配置中选择使用。

完成后确认

  • 屏幕预览中的按钮位置与实际设备相符。
  • 显示配置里已经选中自定义布局。

OLED 布局保存后，还需要在显示配置中启用显示并选择对应布局。

下一步：按键灯光顺序设置

按键灯光顺序设置

设置 RGB 按键灯在灯带链路中的顺序，影响流水、渐变和按键灯效对应关系。

照做

  1. 进入 LED 配置或专业模式灯光工具。
  2. 按视频进入按键灯光顺序设置。
  3. 按实际灯带焊接顺序排列按钮 LED。
  4. 保存后观察灯效是否与实体按键顺序一致。
检查点和注意事项

完成后确认

  • 第一个按钮 LED 从索引 0 开始。
  • 灯效移动方向和实际灯带顺序一致。

灯光顺序错了，颜色本身可能正常，但动态效果会错位。

下一步：按键灯光颜色设置

按键灯光颜色设置

为单个或多个按键设置普通状态和按下状态颜色。

照做

  1. 进入专业模式并选择灯光工具，或进入自定义主题页面。
  2. 选择目标按键。
  3. 设置 Normal Color 和 Pressed Color。
  4. 需要统一颜色时可多选按键批量设置。
检查点和注意事项

剩余动作

  1. 保存并用灯光预览或实体灯检查效果。

完成后确认

  • 松开和按下时颜色符合预期。
  • 颜色没有因为灯光顺序错误映射到其他按键。

如果按键颜色显示到别的按键上，先回到“按键灯光顺序设置”。

下一步：氛围灯配置或预设

氛围灯配置或预设

为外壳或设备内部的 Case RGB/氛围灯添加配置，或套用已有预设。

照做

  1. 进入专业模式或 LED 配置的 Case RGB 区域。
  2. 选择 Off、Ambient 或 Linked 等颜色类型。
  3. 按视频添加氛围灯配置或套用预设。
  4. 选择已有预设，或新增自定义配置。
检查点和注意事项

剩余动作

  1. 保存后观察实体氛围灯。

完成后确认

  • 氛围灯索引位于按钮灯之后，且没有和玩家灯冲突。
  • Ambient 模式下氛围灯可以独立于按钮灯工作。

只想快速使用时优先套用预设；需要细调颜色再看下一步。

下一步：氛围灯颜色设置

氛围灯颜色设置

调整氛围灯的默认颜色或预设颜色。

照做

  1. 选择已经添加的氛围灯配置。
  2. 打开颜色选择器。
  3. 设置空闲颜色或预设颜色。
  4. 保存并确认实体灯颜色。
检查点和注意事项

完成后确认

  • 氛围灯颜色改变的是外壳/背景灯，不是按键灯。
  • 颜色和亮度不会影响按键识别或供电稳定性。

如果灯光数量较多，亮度过高可能增加供电压力；异常时先降低最大亮度。

下一步：自动校准

自动校准

开机或连接后预留约 3 秒初始化时间，自动读取当前磁轴空闲状态并更新校准补偿；用于多平台切换时保证输入稳定，一般保持默认开启。

照做

  1. 进入磁轴配置页面。
  2. 找到自动校准开关。
  3. 普通使用保持开启。
  4. 开机、重启或连接设备后，等待约 3 秒初始化完成，再开始测试按键。
检查点和注意事项

剩余动作

  1. 这个等待用于应对 PC、主机、Switch、XInput 等多平台切换场景，让设备先稳定磁轴基线和输入识别。
  2. 如果要做特殊调试、固定手感，或确认自动校准影响使用，再考虑关闭。
  3. 保存配置。

完成后确认

  • 设备刚连接时如果比普通模式多等待约 3 秒，属于重新获取磁轴状态的正常现象。
  • 初始化完成后，按键静止时没有误触、断触或异常跳动。

目的说明：自动校准会让开机/连接初始化增加约 3 秒，用这段时间读取当前磁轴空闲点并更新补偿，主要是为了应对同一设备在 PC、主机、Switch 等平台之间切换后环境和识别状态变化，减少漂移、误触、断触或第一次连接不稳定的问题。
